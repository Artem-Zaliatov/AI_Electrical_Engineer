Что происходит: git status показывает изменения, git add . 
добавляет все изменённые и новые файлы в будущий коммит, 
git commit сохраняет текущую версию проекта локально в Git, 
а git push отправляет коммит в GitHub

git status
git add .
git commit -m "Complete Section 1 main dimensions calculation"
git push
